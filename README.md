# Plantivity: Tkinter Task Manager

Plantivity is a Python-based task management application built with Tkinter. The app helps users create, organize, and track daily tasks while growing a virtual garden as tasks are completed.

## Project Overview

Plantivity was created as a group project for CCT211: Fundamentals of User Interface Programming at the University of Toronto Mississauga. The goal of the project was to design a user-friendly desktop application that supports productivity, task organization, and motivation through an interactive visual tracker.

Unlike many online task management tools, Plantivity focuses on local storage, allowing users to manage their tasks without relying on cloud-based accounts.

## Features

* User-friendly Tkinter interface
* Task creation and organization
* Ability to add, edit, and delete tasks
* Task details such as title, description, category, due date, priority, and completion status
* Visual plant growth tracker based on completed tasks
* Local data storage
* Interactive productivity system that rewards users as they complete tasks

## How to Run

1. Download or clone this repository.
2. Make sure Python is installed.
3. Install Tkcalendar if needed:

```bash
pip install tkcalendar
```

4. Run the program:

```bash
python main.py
```
